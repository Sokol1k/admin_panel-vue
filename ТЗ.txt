Admin Panel - Vue.js
Страницы:
•	авторизация: логин, регистрация. После регистрации выводить сообщение о том что юзера должен проапрувить админ
•	Dashboard: страница с графиком, который показывает количество созданных бизнесов по дням.
•	Businesses: страница с таблицей бизнесов и кнопкой добавления нового бизнеса. Должна быть возможность фильтровать, сортировать бизнесы. В таблице должны быть все данные которые возвращает апи + колонка actions c иконками просмотра, редактирования, удаления и добавления в список сравнения бизнесов
•	страница создания нового бизнеса, со всеми полями которые требует апи.
•	страница самого бизнеса, с возможностью его редактировать. На этой странице отображаем все поля которые возвращает апи
•	Users: страница со списком всех юзеров. Эту страницу видит только администратор. Тут можно подтверждать существующих юзеров
•	Compare: страница со списком добавленных в сравнение бизнесов. Если списком добавленных в сравнение бизнесов пустой то должен отображаться инпут с поиском по названию бизнеса(этот инпут не обязательный для выполнения, но его  выполнение будет большим плюсом в оценке задания)

Технологии:
•	vue + vuex. Использовать state.
•	для работы с апи нужно использовать axios, а также нужно написать класс клиента, который будет автоматически обрабатывать rest api json ответы, и добавлять заголовки авторизации.
•	для UI использовать набор ADMIN TEMPLATE основанный на vuetify, шаблон в аттач
•	для сборки проекта использовать webpack. лучше всего использовать Vue CLI
•	весь код должен быть структурирован по логическим блокам и комментирован.
•	для графиков использовать highcharts, amcharts, chart.js или другую библиотеку для графиков на ваше усмотрение.
